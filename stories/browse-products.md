# Browse Products

## User Story
As a shopper, I want to browse the product catalogue, search for products by name, and filter by category, so that I can find items I want to purchase.

## Acceptance Criteria

1. **View all products**: When I visit the homepage, I should see all 18 products displayed in a grid layout.
2. **Search by name**: When I type a product name in the search bar, only matching products should be visible. The results count should update.
3. **Search with no results**: When I search for a term with no matches, a "no results" message should appear.
4. **Filter by category**: When I click a category filter chip (Tech, Home, Fashion, Beauty, Fitness, Food & Drink), only products in that category should be visible.
5. **Active filter indicator**: The selected category chip should be visually highlighted (active state).
6. **Reset filters**: When I click "All", all products should be visible again.
7. **Combined search and filter**: Search should work within the selected category filter.
