As a product owner I would like the user to be able to select one  product, add it to the cart and checkout
## Acceptance criteria

- User should be able to click add button on any one  product .
- User can verify the same product is added in Cart 
- Verify the Quantity and price of the product added to cart
- User should be able to checkout by pressing Checkout button
- User should be able to see the choose delivery button
- Clicking choose delivery should open up choose delivery options 
- The two options are Standard shipping and Store pick up
- User will choose Standard shipping by default 
- User should be able to click Continue payment button
- User should be able to press review order button 
- User should be able to click Place order button 
- User should be able to see Order Confirmed!
- Order confirmation page should have Continue shopping button 
- Pressing continue button should dismiss the order confirmation page .
